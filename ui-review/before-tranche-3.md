# Tranche 3 Before Audit (Care Tasks + Photos)

## Scope
- Care Tasks (`CareTaskListView`)
- Photo Gallery (`PhotoGridView` / `PhotoGalleryContainerView`)
- Visual references: `ui-review/before-detail/care-tasks-t3.png`, `ui-review/before-detail/photos-t3.png`

## Findings
- Care task screen has strong data grouping but weak top-level orientation and icon-only toolbar controls.
- Search and chips are functional but not visually anchored to a clear page summary/context block.
- Photo grid is efficient but lacks a premium top-level frame; add-action tile appears utility-first rather than polished.
- Selection mode actions are functional but visual hierarchy between primary and secondary controls can be improved.
